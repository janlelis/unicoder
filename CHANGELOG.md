## CHANGELOG

### 1.0.0

With the first 1.0 release, unicoder supports 10 indexes:

- blocks
- categories
- confusable
- display_width
- emoji
- name
- numeric_value
- scripts
- sequence_name
- types

All indexes can be build in `marshal` format (Ruby's internal
serialization format) and some now support `esm` (JavaScript module)
